How this works

Activity recognition is useful in many domains including biometrics, video- surveillance, human-computer interaction, assisted living, sports arbitration, in-home health monitoring, etc. Yoga posture recognition is a relatively newer application.
The user of our system need to be present in front of the webcam and should follow the instruction provided by the system. The pose of the user is compared with the target pose and difference in coordinated of various body joints are checked to provide corrections. This project lays the foundation for building such a system by using deep learning approaches to accurately detect yoga poses in real-time video.
